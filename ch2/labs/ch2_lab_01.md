## 🎯 主題: 變數宣告與作用域

- 了解變數宣告的正確方式

- 認識 strict mode 的好處
- 發現未宣告變數的風險

### 🏪 商業情境

某線上商店要計算訂單總金額：

小計 = price × quantity
折扣後金額 = 小計 - discount

------------------------------------------------------------------------

## Part 1：找出問題

請閱讀以下程式：

``` js
function calculate(price, quantity, discount) {
    total = price * quantity;

    if (discount) {
        var total = total - discount;
    }

    return total;
}

console.log(calculate("100", 2, 50));
```

### 請回答：

1.  是否有未宣告變數？
2.  `var total` 是否只存在於 `{}` 區塊內？
3.  如果使用 `"100"`（字串），會發生什麼事？

------------------------------------------------------------------------

## Part 2：立即修正

請依照以下規則改寫：

-   加入 `"use strict";`
-   不可使用 `var`
-   所有變數必須先宣告
-   使用清楚的變數名稱

請完成：

``` js
"use strict";

// 在此改寫你的版本
```
