# Lab 05-3

**預估時間：** 15 分鐘

## 情境

你正在維護一個電商平台的訂單報表功能。

系統中有一份訂單金額資料，工程師希望完成以下工作：

- 找出所有 VIP 訂單（訂單金額大於 1000）
- 將每筆訂單標示為 `VIP` 或 `一般`
- 算出 VIP 訂單的總金額
- 將所有訂單依照金額由小到大排序

但是目前的程式碼有兩個問題：

1. 排序結果可能不正確
2. 程式寫法可以更清楚、更符合各種陣列迭代方法的用途

---

## Starter Code

```js
let orders = [1200, 80, 1500, 300, 2000, 500];

// 1. 排序
let sortedOrders = [...orders];
sortedOrders.sort();

// 2. 找出 VIP 訂單
let vipOrders = [];
for (let i = 0; i < orders.length; i++) {
  if (orders[i] > 1000) {
    vipOrders.push(orders[i]);
  }
}

// 3. 產生標籤
let labels = [];
for (let i = 0; i < orders.length; i++) {
  if (orders[i] > 1000) {
    labels.push(`${orders[i]}(VIP)`);
  } else {
    labels.push(`${orders[i]}(一般)`);
  }
}

// 4. 計算 VIP 總金額
let vipTotal = 0;
for (let i = 0; i < vipOrders.length; i++) {
  vipTotal += vipOrders[i];
}

// 5. 印出結果
labels.forEach(label => console.log(label));
console.log('VIP total:', vipTotal);
console.log('sorted:', sortedOrders);
```

## 任務

請修改程式碼，完成以下要求：

1. 讓 `sortedOrders` 真的依照數字由小到大排序
2. 使用 `filter()` 取得所有 VIP 訂單
3. 使用 `map()` 產生 `labels`
4. 使用 `reduce()` 算出 `vipTotal`
5. 保留 `forEach()` 來印出每個 label

---

## 提示

- 數字排序通常需要提供比較函數
- `filter()` 用來過濾元素
- `map()` 用來將元素轉換成新值
- `reduce()` 用來把多個值歸納成一個值
- `forEach()` 適合對每個元素執行輸出或副作用操作
