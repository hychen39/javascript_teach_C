# 🧪 Chapter 5 課堂練習：Map 物件（10 分鐘）

## 🎯 練習目標

- 熟悉 Map 的基本操作（set / get / has / 遍歷）
- 理解 Map 作為「查找與統計工具」的用途
- 建立 Map 與 Object 的使用區別

---

## 📘 題目：購物車商品統計（Map 實作）

某電商系統有以下「購物紀錄」，每筆資料代表一個商品被加入購物車：

```js
const cartLogs = [101, 102, 101, 103, 102, 101];
```

---

## 🧩 任務

### 統計每個商品出現的次數

👉 輸出結果（順序不限）：

```
101 → 3 次
102 → 2 次
103 → 1 次
```

---

## ⛳ 限制條件

- 必須使用 `Map`
- 不可使用 Object `{}` 當作統計工具
- 必須使用：
  - `set()`
  - `get()`
  - `has()`

---

## 💡 提示

```js
if (map.has(key)) {
    map.set(key, map.get(key) + 1);
} else {
    map.set(key, 1);
}
```
